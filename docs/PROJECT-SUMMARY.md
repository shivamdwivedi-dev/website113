## 💡 Customization Guide

### Change Colors
Edit `common.css` and update CSS custom properties:
```css
--primary-color: #d4af37;  /* Change this */
--secondary-color: #ff6b6b; /* Or this */
```

### Change Fonts
Edit `common.css` and update font-family:
```css
font-family: 'Poppins', sans-serif;  /* Change to any Google Font */
```

### Change WhatsApp Number
Edit `assets/js/upgrades.js` line 70:
```javascript
button.href = 'https://wa.me/YOUR_PHONE_NUMBER';
```

### Change Dark Mode Colors
Edit `assets/css/upgrades.css` and find `body.dark-mode`:
```css
body.dark-mode {
  --primary-color: #f0f0f0;  /* Light text */
  --bg-light: #1a1a1a;       /* Dark background */
}
```

## 🆘 Troubleshooting

| Problem | Solution |
|---------|----------|
| **Emails not arriving** | Check FormSubmit email verification |
| **Website not loading** | Check file paths are correct |
| **Dark mode not saving** | Browser privacy might block localStorage |
| **Images not showing** | Verify image file paths in HTML |
| **Animations laggy** | Disable browser extensions, try different browser |
| **Mobile looks weird** | Check viewport meta tag in HTML head |
| **WhatsApp link not working** | Desktop needs web.whatsapp.com access |
| **Form won't submit** | Check browser console for JavaScript errors |
