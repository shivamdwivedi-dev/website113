## Quick Customization

### Change WhatsApp Number
1. Open `assets/js/upgrades.js`
2. Find line: `button.href = 'https://wa.me/919876543210...`
3. Change `919876543210` to your number
4. Save and refresh browser

### Change Colors
1. Open `assets/css/upgrades.css`
2. Find color codes: `#d4af37` (gold), `#ff6b6b` (red), `#25d366` (green)
3. Replace with your colors
4. Save and refresh browser

## Troubleshooting

**Q: Back-to-Top button not showing?**
A: Scroll down at least 300px on the page

**Q: Dark mode not saving?**
A: Browser privacy settings might block localStorage. Try normal browsing mode.

**Q: WhatsApp link not working?**
A: Desktop opens WhatsApp Web at web.whatsapp.com. Make sure logged in there.

**Q: Lightbox not opening?**
A: Make sure you're clicking on images with correct CSS classes
