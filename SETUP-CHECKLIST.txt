🆘 TROUBLESHOOTING
================================================================

Problem: Form not sending emails
Solution: Check FormSubmit form action in contact.html
         Verify email address: shivamdwivedi280708@gmail.com

Problem: Animations not showing
Solution: Ensure assets/css/animations.css is linked
         Check assets/js/form-handler.js is loaded

Problem: Website looks broken on mobile
Solution: Check browser zoom (should be 100%)
         Try different browser
         Clear browser cache

Problem: Images not loading
Solution: Ensure images are in assets/images/ folder
         Check file paths in HTML
         Use correct image extensions (.jpg, .png)

Problem: Links not working
Solution: Verify all .html files in root directory
         Check link paths (case-sensitive)
         Test in different browser
